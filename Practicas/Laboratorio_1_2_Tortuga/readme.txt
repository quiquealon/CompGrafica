Programa que grafica una tortuga usando circulos programados como poligonos a los cuales
se les da el numero de lados , el radio , y la posicion x y y, para la tortuga se utilizo 6 circulos.

Para mover la tortuga se tiene las siguientes instrucciones a traves del teclado una vez que se 
encuentre corriendo el programa

Translacion

w -> adelante
s -> atras
a -> derecha
d -> izquierda

Rotacion

t -> arriba
g -> abajo
f -> izquierda
h -> derecha

Escalado

= -> positivo
- -> negativo
