Laboratorio 4 - Camaras 

Dotar al programa de una tecla que permita cambiar el modo de
proyección entre ORTOGONAL y PERSPECTIVA.
Programar otros modos de movimiento de cámara como son el MODO PAN
o el MODO TRÍPODE.

Para poder cambiar de camaras se utiliza la Tecla F4 para poder hacer toggle entre ambos metodos

Para poder utilizar el modo pan y tripode se utiliza la Tecla F5 para poder hacer toggle entre ambos metodos
